5명

## 맡은 역할
**1. 통합 게시판 기능**
- 사원뷰 통합게시판(부서별 게시판 - 동적 게시판)
- 사원뷰 공지사항, 관리자뷰 공지사항
- 사원뷰 FAQ, 관리자뷰 FAQ
- 관리자뷰 통합 게시판 관리, 게시판별 작성 수 추이

**2. 사원, 부서관련 기능**
- 사원뷰 사원 정보 수정, 관리자뷰 사원 뷰(같은 뷰에서 등록, 수정 처리)
- 사원뷰 조직도, 관리자뷰 조직도 관리
- 관리자뷰 부서 생성,수정, 삭제
- 사원 주소록 관리(등록, 수정, 삭제)
